What does this project do!?

Schema.org is a nice initiative to produce quality content adhering some standarts which would "make happy" the search crawlers, improve your page rankings (maybe) and bring a smile to people who believe in the Linked Web.

While the Linked Web is a highly discussed topic in the last xx years, the use of content standards, schemas or models is not to the desired level (so it seems to me). This side project is based on Grails (at the moment speaking 2.3.2) and intends to port all the Schema.org entities to a database.
Considering the GORM features of Grails, it should be easy to port this information to MySQL, MongoDB, Postgress and any other SQL Database or NoSQL you please.

SQL Structure

The project aims to provide a structure which can be used in your favorite SQL Database or even NoSQL. The Grails App will allow you to connect your code to a desired Database. From that moment, the initial schema will be created and you can have some simple forms for adding valueable content to your repository

The information on your repository can navigated in a normal fashion and also over a REST interface. Through the Interface, you can have second party application query or update the Repository over REST. Documentation will be provided soon

Most of the credits go to Grails here for their wonderful framework
